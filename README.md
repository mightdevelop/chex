# Chex
Chex is a backend application for playing chess online

## Stack:
- NestJS
- Typescript
- PostgreSQL
- Redis
- Swagger

## How to run
1. Have PostgreSQL run
2. Have Redis run
3. Edit `@.env` file, if needed
4. Run `npm run start:dev` in the root
